# metrics_eval

Metrics is a collection of fast metrics built with [Numba](https://github.com/numba/numba).

Currently it offers high performant implementations of the following ranking metrics:
* hit_list_at_k
* hits_at_k
* precision_at_k
* recall_at_k
* r_precision
* mrr
* average_precision
* map
* dcg
* idcg
* ndcg


## Dependencies
* Numba >= 0.49.1
* Numpy >= 1.15
